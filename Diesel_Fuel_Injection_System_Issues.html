એન્જિન વધુ ગરમ થવું

ઇન્જેક્શન સિસ્ટમ સાથે સીધી રીતે સંકળાયેલ ગંભીર સમસ્યા — ૮૦% કેસ ટાઇમિંગ અથવા ઓવર-ફ્યૂઅલ સ્પ્રેથી થાય

📌 કારણો

ગરમ થવાના મુખ્ય કારણો

વધારે ઇંધણ: ઇન્જેક્ટર ઘણું ઇંધણ છોડે — અધૂરું દહન, ધુમાડાની નળી ખૂબ ગરમ.
ઇન્જેક્શન મોડું: ઓછા દબાણ વખતે ઇન્જેક્ટ, દહન અધૂરું — ગરમી વધે.
નોઝલ છંટકાવ ખરાબ: ઝીણી ઝાકળ ન બને — ઇંધણ ધારામાં, ગરમ ભાગો સ્પર્શ કરે.
પાછું ઇંધણ ગળવું: ઇન્જેક્ટર અંદરથી ઇંધણ પાછું ટાંકીમાં — ઇન્જેક્ટર ગરમ રહે.
🔧 ટેસ્ટ

બૉશ ટેસ્ટ બેન્ચ ચકાસણી

સંપૂર્ણ ભાર ફ્લો
૧૦૦૦ અને ૧૬૦૦ બાર દબાણ પર ઇંધણ ડિલિવરી — બધા ઇન્જેક્ટર ૨–૩% ની અંદર.
છંટકાવ પેટર્ન
ઝીણી ઝાકળ ન હોય — ઓપ્ટિકલ ટેસ્ટ — ન હોય તો નોઝલ બદલો.
પાછું ઇંધણ
પાછા ઇંધણ માપો — બૉશ ૧૦૦ ચળ./cc ધોરણ — વધારે = ઇન્જેક્ટર ગળે.
ઇન્જેક્શન સમય
ટોચ બિંદુ (TDC)ની ગણતરી — ±૦.૫ ડિગ્રી ચોકસાઈ — બૉશ ડેટા શીટ.
✅ પ્રમાણ

અપેક્ષિત પરિણામ

ગરમ એન્જિનમાં ૮૦% કેસ ટાઇમિંગ ખોટું અથવા ઓવર-ફ્યૂઅલ છંટકાવ ખરાબ હોવાથી થાય. ફ્લો ટેસ્ટ અને ટાઇમિંગ ચેક — બંને ભેગા — ચોક્કસ કારણ શોધો.
⚡

ઓછી શક્તિ

એન્જિન ઝડપ ઉઠાવ ન કરે, ભાર ઉઠાવવામાં તકલીફ, ઝડપ વધારવામાં આળસ

📌 કારણો

ઓછી શક્તિના મુખ્ય કારણો

ઇન્જેક્ટર ઓછું ઇંધણ: નોઝલ બંધ, ઘસારો, વાઇન્ડિંગ નબળી — ઇંધણ ઓછું.
પંપ ઓછું દબાણ: સામાન્ય રૂ-રેલ — ૩૦૦–૧૬૦૦ બારની મર્યાદા ન જળવાય.
ઇન્જેક્ટર ચળવળ ઓછી: ભૂંસ ઘસાઈ ગઈ — સ્ટ્રૉક ઓછો.
પ્રારંભ ઇન્જેક્શન નથી: મુખ્ય દહન ઠૂંસાઈ — સરળ શક્તિ ન મળે, ઠક-ઠ.
ઇંધણ ગાળણ ભરેલ: દબાણ ઘટે — ઇન્જેક્ટર સુધી ઇંધણ ઓછું.
🔧 ટેસ્ટ

બૉશ ટેસ્ટ બેન્ચ ચકાસણી

સંપૂર્ણ ભાર માત્રા
ઝડપ-પ્રમાણ ઇંધણ cc/ચળ — બૉશ ડેટા શીટ સામે — ±૨% ભેગ.
ઓછી ઝડપ માત્રા
ઓછી ઝડપ RPM અનુકરણ — ઇન્જેક્ટ થતી ઇંધણ માત્રા — ઓછી = ઠક-ઠ.
પ્રારંભ ઇન્જેક્શન
સામાન્ય રૂ-રેલ ઇન્જેક્ટર — ૧–૨ mg પ્રારંભ ઇન્જેક્શન — ઓછી = ઓછી શક્તિ.
રૂ-રેલ દબાણ
૩૦૦, ૮૦૦, ૧૨૦૦, ૧૬૦૦ બાર — પંપ ડિલિવરી ચકાસો.
નોઝલ ગળવું
૧૦૦ બાર — ૧૦ સેકન્ડ — ટીપું ન આવે = સારું, ટીપું = નોઝલ ખરાબ.
✅ પ્રમાણ

અપેક્ષિત પરિણામ

બધા ઇન્જેક્ટર ±૨% ઇંધણ ભેગ — નહિ તો નબળો ઇન્જેક્ટર શક્તિ સંતુલન ખોટો કરે. ફ્લો ઇમ્બૅલૅન્સ ૫% કે વધારે = ઇન્જેક્ટર બદલો.
🚀

એન્જિન ચાલુ ન થવું

ઠંડા કે ગરમ — બંને પરિસ્થિતિ — ઇંધણ દબાણ ઓછું = ચાલુ ન થાય

📌 કારણો

ચાલુ ન થવાના મુખ્ય કારણો

નોઝલ બેઠક ગળવું: ટોચ પર ઇંધણ ટપકે — રૂ-રેલ દબાણ ઓછો રહે.
નિયંત્રણ વાલ્વ ગળવું: ઇન્જેક્ટર અંદર ઘસારો — ઇંધણ ઘણું પાછું.
ઇન્જેક્ટર બંધ ન થાય: બાકી ઇંધણ સિલિન્ડરમાં — સંકોચ ઘટે.
ઇંધણ વિતરણ વાલ્વ ખરાબ: નળીમાં દબાણ ટકે નહિ — ઇન્જેક્ટ ઓછું.
ખૂલવાનું દબાણ ઓછું: ૨૦૦–૨૫૦ બારની જગ્યાએ ૧૨૦–૧૪૦ — છંટકાવ વહેલો.
🔧 ટેસ્ટ

બૉશ ટેસ્ટ બેન્ચ ચકાસણી

ખૂલવા દબાણ
ઇન્જેક્ટર હાથથી દબાણ — ચોક્કસ બાર — બૉશ ડેટા શીટ સામે (૨૦૦–૨૫૦ bar).
નોઝલ ગળવું
૧૫૦ બાર — ૩૦ સેકન્ડ — સૂકી ટોચ = સારી, ભીની = નામંજૂર.
પાછું ઇંધણ
૧૦૦ ચળ. પછી cc — ધોરણ કરતા વધારે = નિયંત્રણ વાલ્વ ગળે.
વાઇન્ડિંગ પ્રતિકાર
સોલેનોઇડ: ૦.૩–૦.૭ ઓહ્મ — ક્રિસ્ટલ: ખાસ કોરોના ટેસ્ટ કિટ.
વિતરણ વાલ્વ
ઊંધા દબાણ ચકાસો — વાલ્વ બેઠક સારી — મૅકૅનિકલ પંપ.
✅ પ્રમાણ

અપેક્ષિત પરિણામ

ઠંડો ચાલુ ચકાસો: -૧૦°C અનુકરણ — ૨૦૦ બારથી ઓછા ખૂલવા દબાણ = ચાલુ ન થાય ચોક્કસ. નોઝલ ટીપું + ઓછું દબાણ = ૯૦% હાર્ડ સ્ટાર્ટ.
💨

કાળો અને સફેદ ધુમાડો

ધુમાડાનો રંગ જ નિદાનની ચાવી — કાળો ≠ સફેદ, બંનેના અલગ કારણ

📌 કારણો

ધુમાડાના કારણો — રંગ પ્રમાણ

🔴 કાળો ધુમાડો

  • ઇન્જેક્ટર વધારે ઇંધણ — નોઝલ ઘસાઈ, મોટા ટીપા, અધૂરું દહન
  • ઇન્જેક્શન ઝડપ આગળ — ઘણું વહેલું — સિલિન્ડર દબાણ ખોટું
  • ઓછી હવા — ઓછો ઑક્સિજન — ઇંધણ બળ્યા વગર
  • છંટકાવ ટૂટ્યો — ઇંધણ દીવાલ પર ચોંટે

⚪ સફેદ ધુમાડો

  • ઇન્જેક્ટર ટપકવું — ઠંડો સિલિન્ડર — ઇંધણ વરાળ ન થાય
  • ઇન્જેક્શન ખૂબ મોડું — નિકાસ ચળવળ સુધી બળ્યા વગર
  • ઠંડક પ્રવાહી ભળે — હેડ ગૅસ્કેટ ખોટો — વરાળ
  • ઓછી ક્ષમતા ઇંધણ — દહન ધીમું
